# 🗺️ Guia Jampa
> Projeto da disciplina **Itinerário Informativo I** — Bacharelado em Ciência da Computação | UNIPÊ

---

## 📌 Sobre o Projeto

O **Guia Jampa** é uma landing page turística desenvolvida para promover e facilitar o acesso às informações sobre os principais pontos turísticos e locais movimentados de **João Pessoa**, capital do estado da Paraíba, Brasil.

A cidade de João Pessoa — carinhosamente chamada de **"Jampa"** por seus moradores — é conhecida por suas praias paradisíacas, rica história colonial, gastronomia diversificada e pela distinção de ser o **ponto mais oriental das Américas**.

---

## 🎯 Objetivo

Desenvolver uma landing page turística **funcional, visualmente atrativa e responsiva** sobre João Pessoa (PB), reunindo em uma única plataforma intuitiva as principais atrações da cidade — voltada tanto para turistas quanto para os próprios moradores.

---

## 🚀 Como Iniciar o Projeto

### Pré-requisitos

- Python 3.10+
- Git

### Passos

1. **Clone o repositório**
```bash
   git clone https://github.com/seu-usuario/guiajampa.git
   cd guiajampa
```

2. **Crie e ative o ambiente virtual**
```bash
   python -m venv venv
   venv\Scripts\activate
```

3. **Instale as dependências**
```bash
   pip install -r requirements.txt
```

4. **Inicie o servidor**
```bash
   python manage.py runserver
```

5. Acesse em **http://127.0.0.1:8000**
